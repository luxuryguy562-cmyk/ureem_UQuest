## 관리자 설정 반영 흐름

관리자가 미션, 보상, 경제설정, 검 레벨, 히든 확률을 변경하면 다음 순서로 반영한다.

```text
관리자 저장
→ 운영 테이블 갱신
→ 경제/화면 스냅샷 재계산
→ app_config_snapshots.payload 갱신
→ 사용자 화면 반영
```

프론트 컴포넌트는 이 스냅샷 결과만 렌더링한다.
