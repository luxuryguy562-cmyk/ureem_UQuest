# U-Quest 포인트 및 보상 정책

## 1. 기본 원칙

### 포인트 가치

1P = 1원

고정

관리자 수정 불가

### 지급 방식

모든 포인트는 조건 달성 즉시 지급

모든 포인트 지급/차감은 이력에 기록

## 2. 기본 행동 보상

### 출석

| 항목 | 값 |
| --- | --- |
| 지급 조건 | 출석 완료 |
| 지급 포인트 | 300P |
| 제한 | 하루 1회 |

### 학습 완료

| 항목 | 값 |
| --- | --- |
| 지급 조건 | 학습 완료 |
| 지급 포인트 | 300P |
| 제한 | Day당 1회, 하루 1개 커리큘럼만 완료 가능 |

### 퀴즈

| 항목 | 값 |
| --- | --- |
| 지급 조건 | 문제 제출 완료 |
| 지급 포인트 | 문제당 300P |
| 제한 | 재도전 없음 |

퀴즈 포인트는 정답 보상이 아니라 제출 보상이다.

오답이 있어도 제출한 문제 수 기준으로 포인트를 지급한다.

정답률은 포인트가 아니라 퀴즈 티어 산정에만 반영한다.

예시

Day별 퀴즈 문제 수에 따라 지급

예: 3문제 풀이

=
900P 지급

### AX

| 항목 | 값 |
| --- | --- |
| 지급 조건 | AX 사진 업로드 또는 촬영 인증 |
| 지급 포인트 | 500P |
| 제한 | 없음 |

AX는 활용 습관화가 목적이므로

인증 시마다 포인트 지급
