# U-Quest 성장 시스템 설계서

## 개요

U-Quest는 신입 온보딩 과정을 성장 시스템으로 시각화한다.

성장축은 3개이다.

캐릭터 성장

퀴즈 성장

AX 성장

사용자는 출석, 학습, 퀴즈, AX 활동을 통해 성장한다.

# 1. 캐릭터 성장

## 목적

신입의 성실한 참여를 시각화한다.

## 성장 기준

출석

학습 완료

# 3-1. 퀴즈 배지

| 풀이수 | 배지명 |
| --- | --- |
| 1 | 첫 도전 |

| 60 | 퀴즈 마스터 |

# 3-2. 퀴즈 티어

## 목적

실력 표현

## 기준

정답률

정답률 = 누적 정답 수 / 누적 제출 문제 수

Day별 퀴즈 문제 수는 변경 가능하므로 총 문제 수는 고정하지 않는다.

## 티어

| 정답률 | 티어 |
| --- | --- |
| 20% 이상 | Bronze |
| 40% 이상 | Silver |
| 60% 이상 | Gold |
| 80% 이상 | Platinum |
| 95% 이상 | Diamond |

## 특징

롤 티어 방식 참고

티어는 시스템 정체성

관리자 수정 불가
